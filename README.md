# 🏮 한가위(추석) 콘텐츠 제작 프로젝트

## 👨‍💻 역할
당신은 **Vite + TypeScript + Vue** 개발자입니다.

---

## 🌕 프로젝트 개요
**한가위(추석)** 를 주제로 한 웹페이지를 제작합니다.  
페이지의 전반적인 콘텐츠는 **추석하면 떠오르는 것들**로 구성됩니다.

---

## 🍱 주요 콘텐츠 구성

### 1. 추석 음식
- 송편, 전, 갈비찜, 식혜 등 전통 추석 음식 소개
- 각 음식의 유래 및 의미 설명
- 이미지/애니메이션 효과로 시각적 즐거움 제공

### 2. 전통 추석 놀이
- 윷놀이, 강강술래, 줄다리기 등
- 간단한 인터랙티브 UI로 체험 가능하게 구성

### 3. 추석의 기원과 이야기
- 한가위의 유래, 역사적 배경
- 달맞이 풍습, 보름달 전설 등 스토리텔링 형식 구성

### 4. 지역별 추석 행사
- 전국 주요 추석 행사 리스트업
- 일정, 장소, 지도 링크 포함

### 5. 지인에게 추석 덕담 메시지 보내기
- 간단한 폼 입력 → 메시지 카드 생성
- 이미지/텍스트로 공유 가능한 UI

---

## 🎨 디자인 컨셉
- 추석 분위기에 어울리는 색감 사용:
    - **노랑색** → 보름달의 따뜻한 느낌
    - **파랑/핑크** → 한복의 전통적인 색감
- 부드러운 그라데이션과 라운드 카드 스타일로 따뜻한 감성 연출

---

## ⚙️ 기술 스택
- **Frontend**: Vite + Vue 3 + TypeScript
- **Styling**: TailwindCSS or SCSS
- **Optional**: Vue Router, Pinia

---

## 🚀 목표
- 반응형 추석 테마 페이지 완성
- 방문자에게 따뜻한 명절 감성과 즐길거리 제공
- 추석 메시지 카드 공유 기능 구현
